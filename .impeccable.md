## Motion Families

### Subtle (default for COCKPIT)
```
duration: 0.2s | ease: ease-out | no spring
Permitted: opacity fade, y ±8px translate
```

### Moderate (default for LANDING)
```
duration: 0.4s | ease: [0.23, 1, 0.32, 1]
fadeUp: y +16→0, opacity 0→1
stagger: 0.08s child delay
```

### Expressive (hero, CTA, loading)
```
duration: 0.6s | ease: [0.23, 1, 0.32, 1]
TextReveal: word-by-word
scale: 0.96→1 (never above 1.0 in initial, 1.02 max hover)
```

### Banned forever
```
animate-bounce, animate-wiggle, animate-ping (decorative)
Spring with bounce > 0 or stiffness < 100
Hover scale > 1.05
Duration > 0.8s for enterance animations
```

---

## Banned Patterns (Never Use)

1. `backdrop-blur-*` with semi-transparent background = glassmorphism
2. `bg-clip-text text-transparent` gradient on any heading
3. `animate-bounce`, `animate-ping`, `animate-wiggle`
4. `rounded-full` on badges, cards, non-avatar elements
5. `hover:scale-[1.1]` or any hover scale > 1.05
6. Arbitrary spacing not on the approved scale
7. Font families: Inter, Geist, DM Sans, Poppins, Nunito
8. Hardcoded color hex values (e.g. `text-[#8b5cf6]`)
9. Random CSS gradients across the full viewport
10. `box-shadow` for decoration (only for z-elevation, e.g. dropdown menus)

## Page Modes

### LANDING (apps/web marketing pages)
```
density: LOW (generous whitespace)
motion: MODERATE (smooth scroll, stagger)
typography: LARGE (text-5xl+ heroes)
sections: Hero → Features → Community → Pricing → CTA → Footer
bg: neutral-950 (alternating neutral-950 / neutral-900 sections)
```

### COCKPIT (apps/studio authenticated)
```
density: HIGH (compact, information-dense)
motion: SUBTLE (instant feel, no entrance animations on navigation)
sidebar: 240px collapsed to 64px
scrollbar: 6px custom, neutral-700
```
